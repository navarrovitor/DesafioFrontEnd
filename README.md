## Para rodar localmente

Clone o projeto

```bash
  git clone https://github.com/navarrovitor/DesafioFrontEnd
```

Vá até a pasta do arquivo

```bash
  cd DesafioFrontEnd
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```

## Tech Stack
